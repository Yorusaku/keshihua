## 🔵 蓝灯阶段设计提案

**文档路径**：`docs/tdd/DataBuffer/DataBuffer-0-蓝灯设计提案-20260306.md`

### 核心设计要点

1. **绝对纯函数**：纯原生 JS，无 Vue 依赖
2. **O(1) 读写**：基于 `Map<string, IAgvData>` 实现
3. **单例模式**：模块级单例，全局唯一
4. **数据隔离**：`getSnapshot()` 返回浅拷贝

### MVP 边界

- ❌ 不包含 Vue `ref/reactive` 绑定
- ❌ 不包含 WebSocket 连接逻辑
- ❌ 不包含数据持久化
- ❌ 不包含数据过滤/转换
