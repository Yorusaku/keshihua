## 📦 重构交付物

### 1️⃣ 新增组件 (`apps/admin/src/views/components/AgvAddModal.vue`)

**文件路径**：`apps/admin/src/views/components/AgvAddModal.vue`

**组件职责**：独立管理新增车辆的弹窗、表单、校验、提交逻辑

**核心导出**：

```typescript
/**
 * 📌 打开弹窗并重置表单（供外部调用）
 * @description 使用 defineExpose 暴露给父组件
 */
const open = () => {
  addForm.value = {
    id: '',
    x: undefined,
    y: undefined,
    status: undefined,
  };
  emit('update:visible', true);
};

/**
 * 📌 暴露方法供外部调用
 */
defineExpose({
  open,
});
```

**Props**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `visible` | `boolean` | 控制弹窗可见性 |

**Events**：

| 事件名 | 参数 | 说明 |
|--------|------|------|
| `update:visible` | `visible: boolean` | 更新弹窗可见性 |
| `success` | 无 | 新增成功时触发 |

**内部状态**：

| 状态 | 类型 | 说明 |
|------|------|------|
| `addForm` | `Ref<...>` | 表单数据 |
| `formRef` | `Ref<any>` | 表单 ref 引用 |
| `addFormRules` | `Object` | 表单校验规则 |
| `isFormLoading` | `ComputedRef<boolean>` | 表单加载状态 |

**核心逻辑**：

```typescript
/**
 * 📌 表单提交处理
 */
const handleFormSubmit = async () => {
  try {
    // ✅ 校验表单
    await formRef.value.validate();
    
    // ✅ 隐藏弹窗，显示加载状态
    emit('update:visible', false);
    
    // ✅ 提交数据
    mutation.mutate(addForm.value, {
      onSuccess: () => {
        // ✅ 成功后触发 success 事件
        emit('success');
      },
    });
  } catch (error) {
    // ✅ 校验失败
    console.error('表单校验失败:', error);
  }
};

/**
 * 📌 Mutation Hook
 */
const mutation = useAddAgvMutation();
```

## 🔧 技术亮点

### 1️⃣ defineExpose 模式

```typescript
// ✅ 暴露方法供外部调用
defineExpose({
  open,
});
```

**优势**：
- ✅ 父组件可通过 `ref` 直接调用子组件方法
- ✅ 避免使用 `v-model` 或 `props` 传递方法
- ✅ 类型安全（通过 `defineExpose` 显式声明）

### 2️⃣ 单向数据流

```vue
<!-- 父组件 -->
<AgvAddModal
  :visible="modalVisible"
  @update:visible="modalVisible = $event"
  @success="handleAddSuccess"
/>
```

**流程**：
- ✅ 父组件通过 `props` 传递 `visible`
- ✅ 子组件通过 `emit('update:visible')` 更新
- ✅ 父组件通过 `@success` 监听成功事件

### 3️⃣ 组件复用性

```typescript
// ✅ AgvAddModal 可在其他页面复用
<template>
  <AgvAddModal
    ref="addModalRef"
    :visible="modalVisible"
    @update:visible="modalVisible = $event"
    @success="handleSuccess"
  />
</template>

<script setup>
// 任何页面都可以复用此组件
import AgvAddModal from '@/views/components/AgvAddModal.vue';
</script>
```
