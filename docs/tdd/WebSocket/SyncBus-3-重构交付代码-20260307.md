## 📦 重构交付物

### 1️⃣ 新增 Hook (`apps/dashboard/src/composables/useAgvSync.ts`)

**文件路径**：`apps/dashboard/src/composables/useAgvSync.ts`

**核心功能**：

```typescript
/**
 * 📌 跨端通信监听 Composable（高阶抽离）
 */
export function useAgvSync(): void {
  /**
   * 📌 onMounted 钩子：订阅 Admin 广播
   */
  onMounted(() => {
    // ✅ 订阅新车数据（携带严格的 IAgvData 类型断言）
    const unsubscribe = agvSyncBus.subscribeNewAgv((agv: IAgvData) => {
      // ✅ 写入 DataBuffer（单条数据数组）
      DataBuffer.getInstance().pushData([agv]);
    });

    // ✅ 保存 unsubscribe 函数（用于后续清理）
    unsubscribeFn = unsubscribe;
  });

  /**
   * 📌 onBeforeUnmount 钩子：取消订阅
   */
  onBeforeUnmount(() => {
    // ✅ 调用 unsubscribe 清理监听器
    if (unsubscribeFn) {
      unsubscribeFn();
      unsubscribeFn = null;
    }
  });
}
```

**新特性**：

| 特性 | 说明 |
|------|------|
| ✅ 声明式 UI | Dashboard 只需 `useAgvSync()` |
| ✅ 隐藏细节 | 订阅、写入、取消订阅全部隐藏 |
| ✅ 类型安全 | `agv: IAgvData` 严格类型断言 |
| ✅ 生命周期管理 | 自动 onMounted/onBeforeUnmount 处理 |

**使用方式**：

```typescript
// ✅ 重构前（命令式）
import { onMounted, onBeforeUnmount } from 'vue';
import { agvSyncBus, DataBuffer } from '@packages/shared';

let unsubscribe: (() => void) | null = null;

onMounted(() => {
  unsubscribe = agvSyncBus.subscribeNewAgv((agv: IAgvData) => {
    DataBuffer.getInstance().pushData([agv]);
  });
});

onBeforeUnmount(() => {
  if (unsubscribe) {
    unsubscribe();
    unsubscribe = null;
  }
});

// ✅ 重构后（声明式）
import { useAgvSync } from '@/composables/useAgvSync';

// ✅ 一行代码完成所有逻辑！
useAgvSync();
```
